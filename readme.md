## Project Goal
A complete, custom implementation of a **Singly Linked List** data structure using ES6 class syntax.

This project covers all fundamental list operations, including:

* **Manipulation:** `append`, `prepend`, `pop`.
* **Traversal & Access:** `size`, `head`, `tail`, `at`, `contains`, `find`.
* **Extra Credit:** Robust `insertAt` and `removeAt` methods.
* **Display:** Custom `toString()` method for console output.
